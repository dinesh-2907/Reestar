Reestar is a simple single-page restaurant website built using **HTML5** and **CSS3**. The website features a clean and responsive design with four food items displayed in an organized layout. It includes a navigation section, food showcase cards, and a contact form, providing a basic yet attractive restaurant landing page experience.

 Features

* Single-page website design
* Four food showcase cards
* Responsive and user-friendly layout
* Clean and modern interface
* Contact form for customer inquiries
* Built entirely with HTML and CSS

 Technologies Used

* HTML5
* CSS3

 Purpose

This project was created to practice front-end web development concepts, including page structure, styling, responsive layouts, and basic user interface design. It serves as a beginner-friendly project for learning HTML and CSS fundamentals.
